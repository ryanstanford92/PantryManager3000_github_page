# PantryManager3000_github_page

## Installation and Using the Application
 ### Installation:
1.	Update device settings to allow installation of unknown apps
Settings -> Apps -> Special App Access -> Install Unknown Apps -> Allow source to install unknown apps (e.g. web browser used to download file)
2.	Navigate to https://ryanstanford92.github.io/PantryManager3000_github_page/ on your Android device (8.0 minimum)
3.	Click the download link and download the APK
4.	Navigate to Downloads
5.	Select the downloaded APK and install
 
 




### Using the Application:

#### Home Screen
The app starts in the home screen containing three buttons:
 
My Pantry: navigates to the pantry where the user’s tracked items are stored

Grocery List: navigates to a checklist of items marked to be purchased

My Recipes: navigates to a list of the user’s recipes


#### My Pantry Screen	
On the My Pantry screen users can view their tracked pantry items.

Add new items to the pantry by clicking the ‘+’ button located at the bottom of the screen.

Select an item in the list to view more details about that item.

#### Pantry Item Screen
	
The Pantry Item screen shows more details about the selected pantry item.

Edit the details and save updates using the save button on the bottom left of the screen.

To remove an item from the pantry select the ‘x’ button on the bottom right of the screen.

#### Grocery List Screen
	
The Grocery List screen contains items with a quantity on order.

Select items in the list to update the pantry listing and change the order amount.

To add new items click the ‘+’ button located at the bottom of the screen.

The check box can be filled as items are purchased.

The submit button in the kebab menu (three vertical dots at the top right of the screen) updates the pantry quantities of the checked off items and removes those items from the grocery list.


#### My Recipes Screen
The My Recipes screen holds a list of the user’s recipes

To view details or make changes to an existing recipe, select the desired recipe in the list.

To add an additional recipe, select the ‘+’ button located at the bottom of the screen.

The recipe list can be filtered by ingredient using the search button at the top of the screen.

All recipes or a filtered list can be shared using the share button in the kebab menu (three vertical dots at top right of screen).


#### Recipe Detail Screen
	
The Recipe screen contains details about the selected recipe.

On this screen, recipes can be updated and saved using the save button on the bottom left of the screen.

Recipes can be deleted using the ‘x’ button on the bottom right of the screen.


#### Recipe Ingredient Screen

Ingredients can be added using the Add Ingredient button in the center of the screen.

Existing ingredients can be edited or removed by selecting the desired ingredient from the list on the bottom half of the screen.
	
The Ingredient screen contains details about an ingredient.

Ingredient details can be updated and saved using the save button located at the bottom left of the screen.

To delete an ingredient, click the ‘x’ button located at the bottom right of the screen.
